Algunos lenguajes de la web

HTML+CSS+Javascript

  • HTML 5

HTML5(HyperText Markup Language, versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML.

Usa la extension .html

  • CSS 3

Hoja de estilo en cascada o CSS (siglas en ingles de Cascading Style Sheets)es un lenguaje usado para definir y crear la presentacion de un documento estructurado escriuto en HTML

Usa la extension css

  • JavaScript

JavaScript (abreviado comunmente "JS") es un lenguaje de programacion interpretado utilizado en el lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en el interfaz de usuario y paginas web dinamicas

Usa la extension .js
